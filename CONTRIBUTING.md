# Contributing to Awesome AI Agents

Thank you for considering contributing! This project aims to curate the best AI agent collections, skills frameworks, and related resources.

## How to Contribute

### Adding a New Resource

1. **Fork** the repository
2. **Add** your resource to the appropriate section in both `README.md` and `README.zh-CN.md`
3. Follow the existing format:
   - Use a clear, descriptive project name with link
   - Include stars count if notable (e.g., `59k+`)
   - Write a concise one-sentence description
4. **Submit** a Pull Request

### Resource Criteria

- **Agent Personas**: Specialized agent personalities with defined roles, workflows, or deliverables
- **Skills Frameworks**: Composable skill systems that teach agents methodologies (TDD, planning, etc.)
- **Framework Collections**: Curated lists of LLM/agent development frameworks
- **Related**: Tools, standards, translations that support the agent ecosystem

### Format Guidelines

- Keep descriptions to 1–2 sentences
- Prefer official/reliable sources
- Check that links are valid before submitting

### Pull Request Process

1. Update both English and Chinese README files
2. Ensure consistent formatting
3. Add entries in the appropriate category
4. PR title: `Add: [Project Name]` or `Update: [Section]`

## Questions?

Open an issue for discussion.
