# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

This is a documentation/curation project. We don't ship executable code that could have security vulnerabilities in the traditional sense.

If you find:

- **Broken or malicious links** in the curated list
- **Phishing or scam repositories** listed
- **Other safety concerns** about listed resources

Please [open an issue](https://github.com/charlie-cao/agents-awesome/issues) with the tag `security` or contact the maintainers directly.

We will review and remove any unsafe entries promptly.
