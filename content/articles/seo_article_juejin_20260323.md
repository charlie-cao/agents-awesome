## 如何快速上手 AI Agent 开发

### 第一步：选择一个框架

新手推荐从 **Dify** 或 **OpenClaw** 开始，上手门槛最低。

### 第二步：学习 Agent 的核心概念

无论选择哪个框架，都需要理解：
- **工具调用（Tool Calling）**：让 LLM 使用外部工具
- **记忆系统（Memory）**：让 Agent 记住对话上下文
- **规划（Planning）**：让 Agent 能够分解复杂任务

### 第三步：构建你的第一个 Agent

以 OpenClaw 为例：

```bash
# 安装
npm install -g openclaw

# 初始化
openclaw setup

# 启动
openclaw start
```

然后配置你的 SOUL.md，一个专属的个人 AI 助手就完成了。
